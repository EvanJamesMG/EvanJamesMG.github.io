发表于 2016-04-13   |     |   次阅读

最近笔试题目中经常出现涉及到数据库的题目，都是常用的增删查改的命令。这里总结一下MySQL中常用的简单的数据库命令，避免再次遇到时又是大写的懵逼。

数据类型介绍

  • 整数类型：INT(显示宽度)
  • 浮点型：FLOAT，DOUBLE（总长度，小数点后长度）
  • 字符串：CHAR(长度固定)，VARCHAR（长度可变）
  • 字符数据：TEXT 一种特殊的字符串类型，只能保存字符数据，比如新闻的内容
    其他数据类型平时遇到较少，先不做总结。

设置表达的外键
设置外键的原则就是必须依赖于数据库中已经存在的父表的主键;外键可以为空值。外键的作用是建立该表与其父表的关联关系。父表中删除某条消息时，子表中与其对应的信息也必须有相应的改变。

注意：子表的外键关联的必须是父表的主键，而且数据类型必须一致。

完整性约束条件表

约束条件 说明
primary key 标识该属性为该表的主键，可以唯一的标识对应的元组
foreign key 标识该属性为该表的外键，是与之联系的某表的主键
not null 标识该属性不能为空
unique 标识该属性的值是唯一的
auto_increment 标识该属性的值自动增加
default 为该属性值设置默认值
